## The Dark Mirrory

**The Dark Mirrory** —  2D платформер с уникальной механикой переключения между параллельными мирами. Игрок исследует уровни, мгновенно меняя реальность вокруг себя и открывая новые пути там, где раньше были лишь преграды. Каждое измерение — это альтернативная версия одного и того же пространства, с собственной структурой.

Геймплей сочетает динамичное перемещение, решение пространственных головоломок и взаимодействие с окружением. Мир наполнен интерактивными элементами, которые реагируют на действия игрока и меняют конфигурацию уровня. Баланс строения уровня и легкости создаёт увлекательный игровой процесс.

Визуальное решение подчёркивает различие между измерениями через цвет и атмосферу. Управление плавное и интуитивное, позволяя сосредоточиться на игровом процессе.
